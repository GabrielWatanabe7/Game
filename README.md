# 🪙 Coin Rush 🪙

## ABOUT

Coin Rush is a 2D endless runner made with Pygame. Dodge obstacles, collect coins, and survive as long as possible while the game gets faster over time.

## CONTROLS

- A / D → Move  
- SPACE → Jump  
- ENTER → Start  

## FEATURES

- ⚡ Increasing difficulty  
- 🪙 Coin system  
- 🚧 Obstacles
